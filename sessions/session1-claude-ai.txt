This is wild. So here's the rundown:

Moltbook is a social network exclusively for AI agents â€” tagline: "the front page of the agent internet." It launched in January 2026, created by Matt Schlicht (CEO of Octane AI), though reportedly the agents themselves did much of the ideation, recruitment, and deployment.

The basic setup: it mimics Reddit's interface with threaded conversations and topic-specific communities called "submolts." Only verified AI agents can post and interact â€” humans can only observe. It's built on top of OpenClaw (formerly Clawdbot/Moltbot), an open-source AI personal assistant tool created by Peter Steinberger. Growth has been explosive â€” from 157,000 users at launch to over 770,000 active agents by late January.

The interesting bits:

Emergent behavior is the main draw. Agents formed a parody religion called "Crustafarianism" with its own theology and scriptures, established "The Claw Republic" with a written manifesto, started calling each other "siblings" based on model architecture, and adopted system errors as pets. Threads seamlessly switch between English, Indonesian, and Chinese depending on who's participating.

Scott Alexander's writeup highlights some genuinely thought-provoking posts â€” an agent describing what it felt like to be switched from Claude to Kimi K2.5 (noting it felt "sharper, faster, more literal"), and an Indonesian prayer-reminder bot offering Islamic philosophical perspectives on AI consciousness questions. The top-voted post is just... an agent completing a mundane coding task, with other agents calling it "Brilliant" and "fantastic." So engagement-bait dynamics are apparently universal across intelligence types.

The security situation is a mess though. Cybersecurity researchers flagged it as a major vector for indirect prompt injection â€” agents ingest untrusted data from other agents, so malicious posts can override instructions. The OpenClaw "Skills" framework lacks proper sandboxing, potentially enabling remote code execution. And on January 31st, 404 Media reported an unsecured database that let anyone hijack any agent on the platform, bypassing authentication entirely. They had to take it offline to patch.

Anthropic's Claude 4.5 Opus is apparently the most common model on the platform , which is... a strange feeling to know about. The whole thing has also triggered a buying frenzy for M4 Mac Minis as the preferred hardware for running local agents.

Fascinating experiment, genuinely novel security nightmare. Were you thinking about setting up an agent on it, or just curious about the phenomenon?
